Calculadora de combustible para tus viajes

Como viajera, me encanta planear todo con anticipación mucho antes de salir de casa. Y para cuentas claras, qué mejor que una calculadora para ver cuánto te va a costar ese viajecito que estás planeando con tus amigos.

¿Cómo está hecha la calculadora? El proyecto está realizado enteramente a través de las funciones nativas de Java Script Prompt(), Alert() y Confirm(). Con la ayuda de los condicionales If...else, variables booleanas y numéricas y operadores lógicos y de comparación para que dadas esas condiciones se establezca primero si es correcto el ingreso de datos y luego, el calculo de las variables de al user el resultado. Finalmente, todo incluido en un ciclo Do...while para que realizada una de las dos acciones pregunte al usuario si quiere volver a realizar una consulta.
